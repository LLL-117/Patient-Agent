PATIENT AGENT

面向患者信息检索的 Query 页面

须填写 11 位手机号（必填，示例见占位符）。输入自然语言问题，可选上传图片。查看模型回答、结构化检索结果；下方「声音选择」选「无」则不生成语音。勾选「Planner 调试」可额外请求 ReAct 规划轨迹（与主问答并行，耗时更长）。

示例问题

请帮我查询患者 P1001 最近一次就诊的要点。
我是 P1001，请总结我最近一次心内科复诊的重点，并给出用药提醒。
请结合上传的图片判断这是否为患者最近的检查资料，并提取要点。
